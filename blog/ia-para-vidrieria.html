El negocio de la vidrería: urgencias y proyectos a medida

Una vidrería maneja dos tipos de pedidos muy distintos. Por un lado, las urgencias — vidrios rotos que hay que reemplazar hoy. Por otro, los proyectos — mamparas de baño, escaleras de vidrio, cortinas de vidrio para locales, espejos decorativos. Ambos tipos requieren atención rápida, pero el proceso de cotización es muy diferente.

Para las urgencias, el cliente quiere respuesta inmediata y un técnico disponible hoy o mañana. Para los proyectos, necesita mediciones, opciones de materiales y un presupuesto detallado. Un empleado de IA como TamoWork puede manejar ambos flujos de forma inteligente, según el tipo de consulta que recibe.

Flujo de atención para urgencias

Cuando el cliente escribe "se me rompió una ventana, ¿pueden venir hoy?", el empleado de IA responde de inmediato con empatía y utilidad:

  1. Confirma disponibilidad para urgencias del día
  2. Pregunta dimensiones aproximadas del vidrio o pide que el cliente envíe una foto
  3. Da un precio de referencia según el tipo de vidrio y el tamaño
  4. Agenda la visita técnica para medición e instalación
  5. Envía los datos de tu empresa para que el cliente esté tranquilo

Todo en menos de 2 minutos. Sin que tú tengas que interrumpir el trabajo que estás haciendo en el local.

Flujo de cotización para proyectos

Para proyectos más grandes, el empleado de IA inicia un proceso de recolección de información:

  • Tipo de proyecto: mampara, espejo, cerramiento, mesa, escalera
  • Medidas aproximadas o si se necesita visita para medición
  • Tipo de vidrio preferido: templado, laminado, satinado, espejado
  • Instalación incluida o solo suministro
  • Plazo estimado para el proyecto

Con esa información, tú puedes preparar el presupuesto mucho más rápido cuando revisas los mensajes. El cliente ya tiene todo el contexto y tú no pierdes tiempo en preguntas de seguimiento.
