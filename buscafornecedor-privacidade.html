Política de Privacidade — BuscaFornecedor

Última atualização: 31 de março de 2026

O BuscaFornecedor (aplicativo Android e site buscafornecedor.com) está comprometido em proteger a privacidade dos seus usuários. Esta política descreve quais dados coletamos, como os usamos e como os protegemos.

1. Dados coletados

  • E-mail e senha — para criação e acesso à conta do usuário.
  • Dados de uso — buscas realizadas e categorias visitadas, usados para melhorar os resultados.
  • Dados de pagamento — processados exclusivamente pelo Mercado Pago. O BuscaFornecedor não armazena dados de cartão de crédito.

2. Como usamos os dados

  • Autenticar o acesso do usuário à plataforma.
  • Processar e confirmar assinaturas.
  • Melhorar os resultados de busca e recomendações de fornecedores.
  • Enviar notificações sobre atualizações do serviço (pode ser cancelado a qualquer momento).

3. Compartilhamento de dados

Não vendemos dados pessoais. Compartilhamos apenas com:

  • Mercado Pago — processamento de pagamentos.
  • Supabase — armazenamento seguro de dados em banco de dados.
  • Autoridades quando exigido por lei.

4. Segurança dos dados

Todos os dados são armazenados com criptografia. Senhas são armazenadas com hash seguro (bcrypt). A comunicação entre o app e nossos servidores é feita exclusivamente via HTTPS.

5. Direitos do usuário (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais.
  • Corrigir dados incorretos.
  • Solicitar exclusão da conta e de todos os dados associados.
  • Revogar consentimento a qualquer momento.

Para exercer seus direitos: contato@buscafornecedor.com

6. Retenção de dados

Os dados são mantidos enquanto a conta estiver ativa. Após solicitação de exclusão, os dados são removidos em até 30 dias.

7. Cookies e rastreamento

Usamos cookies apenas para manter a sessão do usuário ativa. Não utilizamos cookies para publicidade ou rastreamento de terceiros.

8. Público-alvo

O BuscaFornecedor é destinado exclusivamente a maiores de 18 anos. Não coletamos dados de menores de idade.

9. Contato
