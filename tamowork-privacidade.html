Política de Privacidade — TamoWork

Última atualização: 31 de março de 2026

O TamoWork (aplicativo disponível em tamowork.com) é um assistente de inteligência artificial que responde automaticamente mensagens do Instagram e WhatsApp diretamente no computador do usuário. Estamos comprometidos em proteger a privacidade dos nossos usuários.

1. Dados coletados

  • Dados de instalação — o aplicativo é instalado localmente no computador do usuário (Windows). Nenhum dado é enviado para servidores externos durante o uso normal.
  • Credenciais de redes sociais — o login no Instagram e WhatsApp é feito diretamente pelo usuário no app. As credenciais ficam armazenadas localmente no dispositivo e não são transmitidas para nossos servidores.
  • Dados de suporte — caso o usuário entre em contato via e-mail ou WhatsApp, podemos armazenar nome e e-mail para responder ao atendimento.

2. Processamento local

O TamoWork utiliza um modelo de inteligência artificial que roda 100% localmente no computador do usuário. As mensagens respondidas pelo app não são enviadas para nenhum servidor externo. Todo o processamento ocorre no próprio dispositivo.

3. Como usamos os dados

  • Permitir o funcionamento do assistente de IA no dispositivo do usuário.
  • Responder solicitações de suporte técnico.
  • Enviar atualizações do software quando disponíveis (apenas se o usuário optar por receber).

4. Compartilhamento de dados

O TamoWork não vende nem compartilha dados pessoais com terceiros. Por ser um aplicativo local, a maior parte dos dados nunca sai do computador do usuário.

5. Segurança

Os dados armazenados localmente pelo aplicativo ficam protegidos pelas permissões do sistema operacional do usuário. Recomendamos manter o sistema operacional e o antivírus atualizados.

6. Direitos do usuário (LGPD)

Conforme a Lei nº 13.709/2018 (LGPD), você tem direito a:

  • Acessar e corrigir seus dados pessoais.
  • Solicitar a exclusão de dados em nosso poder.
  • Revogar o consentimento a qualquer momento.

Para exercer seus direitos: contato@tamowork.com

7. Retenção de dados

Dados de suporte (e-mail/nome) são mantidos pelo tempo necessário para resolver a solicitação. Após isso, são excluídos. Os dados locais do app podem ser removidos pelo próprio usuário desinstalando o aplicativo.

8. Público-alvo

O TamoWork é destinado a donos de pequenos negócios maiores de 18 anos. Não coletamos dados de menores de idade.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. Alterações significativas serão comunicadas via e-mail ou pelo site tamowork.com.
